    <!doctype html>: Es una declaración que especifica la versión del lenguaje HTML utilizada en el documento.

    <html lang="en">: Define el elemento raíz del documento HTML y especifica el idioma principal del contenido (en este caso, en inglés).

    <head>: Contiene metadatos y enlaces a recursos externos utilizados por la página.

    <meta charset="utf-8">: Especifica la codificación de caracteres utilizada en el documento (UTF-8 en este caso, que es ampliamente compatible y admite múltiples idiomas).

    <meta name="viewport" content="width=device-width, initial-scale=1">: Define las propiedades de visualización de la página en dispositivos móviles.

    <title>ViaUy</title>: Establece el título de la página que se mostrará en la pestaña del navegador.

    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-9ndCyUaIbzAi2FUVXJi0CjmCapSmO7SnpJef0486qhLnuZ2cdeRhO02iuK6FUUVM" crossorigin="anonymous">: Es un enlace al archivo CSS de Bootstrap que se utilizará para aplicar estilos a la página.

    Este fragmento representa una barra de navegación responsiva utilizando los componentes de navegación de Bootstrap.

    <nav class="navbar navbar-expand-lg bg-body-tertiary">: Define la barra de navegación y aplica estilos específicos de Bootstrap (bg-body-tertiary).
    
    <div class="container-fluid">: Envuelve el contenido de la barra de navegación para mantenerlo dentro de un contenedor fluido.
    
    <a class="navbar-brand" href="#">ViaUy</a>: Es el logotipo o nombre de la marca que se muestra en la barra de navegación.
    
    <button class="navbar-toggler" ...>: Es el botón de alternancia que se muestra en pantallas más pequeñas para expandir o contraer el menú de navegación.
    
    <div class="collapse navbar-collapse" id="navbarNavDropdown">: Envuelve el menú de navegación para controlar su visibilidad en pantallas más pequeñas.
    
    <ul class="navbar-nav">: Contiene la lista de elementos de navegación.
    
    <li class="nav-item">: Representa un elemento de navegación.
    
    <a class="nav-link ...>: Es el enlace o el elemento de menú de navegación.
    
    <li class="nav-item dropdown">: Representa un elemento de navegación que despliega un menú desplegable.
    
    <ul class="dropdown-menu">: Contiene los elementos del menú desplegable.
